# Setup 

To setup :
1. Follow README.md or Ask Chatgpt "How can I setup github.com/Codeotus/Codeotus ?"

# Launch

Launch with `nv` or any custom alias and you will see a minimal dashboard , try navigating with the buffer navigation.

# Open a file 


Open a file with : 

```sh
nv python.py
```
# Press :

`i` to enter insert and write something.
`esc` key to enter normal mode.
`:` to enter command mode.
`:!` to enter shell command line mode.
`space` bar or key to open leader menu.
`space + t` or any like 'space with a or b or ... '

After doing this all you will see some more things you can proceed like :
`space + tgn` this will toggle line number.

`space + tg` will open toggle leader keys use `backspace` key to go back to leader t cateogry and press t again to come back to leader tg group.

`space + ctrl d/ ctrl u` will go down or up respectively.

# Exit 

`:q` if has written and saved something.
`space + qq` if has written and saved something.
