# Terminal 
Press ctrl + space to open terminal
Press esc key 2-3 times very fast to exit terminal mode.
Press qq or ctrl + space to exit or toggle respectively.

# Git 
Press Leader gg to open Lazy git
Make sure you hve Lazy git downlaoded in your terminal.
Follow Lazy git tutorials to use it.

# File tree
Space + e to toggle file tree.
Follow file tree tutorial to use it.

# Toggle :
Autosave with : space + as
Undo tree with : space + ut 
Lsp poup with : space + tp
Take a look at : space + tg 
Take look at : Space + ct 
Take look at : Space + t

# Autosave :
By default is set to true & may sometimes feel too restrective to undo and it is better to turn of while undoing with leader as or use undo tree.

# Undo tree :
Use leade ut to toggle undo tree and it will give you your undo history.

# History :
Press leader yh to see yank history.
Press leader nh to see message history.

# Open File Organizer with :
Space + ao + f or anything you like.

# Session :
Add session with Leader ss.
Pick session with Leader sf.
Load last session with Leader sl.

# Buffer :
Shift + h or Shift + l to navigate buffer files left or right repsectively.

Leader + bd = delete buffer from current session.

# Oil :
Press : `-` in normal mode
Follow oil tutorials to use it.

# Quit with :

Leader q series.
Leade w series.

# Save with :
Leader ws , wa ,bs or even use leader as to toggle autosave.

# Reload file :
Leader cfr or leader reb.

# Lsp :
Use K without any space in nomal mode to see docs of the word under the cursor.
Use Normal mode + cursor under underlined word to see the errors.
Use leader cD to turn docs on/off for the completion float.

# Change theme :
Leader tft. 
:colorscheme theme_name.

Do a permanent setup with : 
:SGT theme_name
